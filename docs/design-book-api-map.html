How to read this

The wiring convention

Each section pairs a re-creation of one UI region with a table. Numbered markers (1 …) on the mock-up point to the matching row. A row names the visible element, the endpoint that serves it, and the request param or response field involved.

GET Read endpoint — lists, lookups, detail, config resolution.
POST Write endpoint — catalog ingest / sync only.
?param Request query/path parameter the control sets.
field.path Response field the control renders.

All 11 endpoints

Complete surface of the /design-book module. Every route is JWT-guarded and documented with an @ApiOperation in Swagger.
