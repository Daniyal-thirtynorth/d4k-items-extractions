2) One tab label has HTML inside it.
   The "Inspiration" tab label came out as raw SVG markup (the icon code) instead of plain text.
   It should just be "Inspiration". Please strip all HTML from every tab label.

   The top of the detail screen has Width / Height / Depth / Programme pills. Clicking a pill opens
   another unit (another code) — for example, on TK6080BZ2, clicking Width 70 opens TK7080BZ2.
   The export does not include this yet. Please add a "configure" object per unit with each pill's
   target code (details in the schema).
